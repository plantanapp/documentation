Generate a SHA-256 hash from a string.

## Parameter Reference
| Parameter | Description | Supports Tokens | Default |
| -- | -- | -- | -- |
| Data To Hash | The data that will be hashed. | No | None |
| Hash Token Name | Token name where the hash will be stored. | No | None |
