## Parameter Reference
| Parameter | Description | Supports Tokens | Default |
| -- | -- | -- | -- |
| User Identifier | The value can be an User Id, User Name or an Email. Supports context tokens and My Tokens. | Yes | None |
| Result Token Name | The result of the user unlock will be set here. Possible values: true, false. | No | None |
